# Implementaci-n-de-una-red-segura-Cliente-Servidor-con-Fedora-Debian-y-Automatizaci-n.
Proyecto en Fedora y Debian que implementa una red segura cliente-servidor con automatización. Incluye configuración de entorno, administración de usuarios, gestión de kernel y sistemas de archivos, respaldos, bitácoras y monitoreo para garantizar seguridad y eficiencia.
